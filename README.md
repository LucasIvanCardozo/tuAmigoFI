# TuAmigoFI

Plataforma colaborativa diseñada para estudiantes de la **Facultad de Ingeniería de Mar del Plata**. 

TuAmigoFI es un espacio donde los alumnos pueden compartir material académico como trabajos prácticos, exámenes parciales y finales, respuestas y resoluciones. Los estudiantes tienen la posibilidad de comentar, evaluar y discutir cada aporte, generando un ciclo de retroalimentación que enriquece el aprendizaje colectivo.

El objetivo principal es construir una comunidad donde el conocimiento fluya entre pares, facilitando el acceso a recursos de estudio y promoviendo la colaboración como motor de aprendizaje.
